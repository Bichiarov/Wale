# Wale Solutte Automação Comercial - Site Institucional

Site estático pronto para publicação no GitHub Pages, Netlify, Vercel ou hospedagem convencional.

## Identidade visual

O site foi criado respeitando a identidade visual da logo da Wale Solutte: fundo preto, azul elétrico, ciano, prata/metálico e branco.

## WhatsApp automático

O botão de WhatsApp troca automaticamente conforme o horário de São Paulo:

- Comercial / Suporte: (13) 98143-4489
- De Segunda à Sexta, das 9h às 18h
- Suporte (Plantão após as 18h): (13) 99692-4357

A regra está no arquivo `assets/js/main.js`.

## Como publicar no GitHub Pages

1. Crie um repositório no GitHub.
2. Envie todos os arquivos deste pacote para a raiz do repositório.
3. Vá em Settings > Pages.
4. Em Branch, escolha `main` e `/root`.
5. Salve e aguarde o link do GitHub Pages.
